## Demo en línea

[Ver proyecto en GitHub Pages](https://adol-33.github.io/Desafio-e_commerce/)

## Descripción
Sitio web de e-commerce responsivo que utiliza CSS Grid y Flexbox para organizar la estructura principal y los productos. Incluye un menú de categorías, una sección de productos y un diseño adaptable a dispositivos móviles.

## Funcionalidades
- Diseño principal con CSS Grid para header, aside, main y footer.
- Productos organizados con Flexbox y tarjetas responsivas.
- Menú de categorías en el aside.
- Imágenes de productos adaptables al contenedor.
- Efectos hover en tarjetas y botones.
- Adaptabilidad a diferentes tamaños de pantalla (media queries).

## Cómo visualizar el proyecto
1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador.
